ETFormValidation

Easy to use jQuery plugin for validating forms.

Download this project as a .zip file Download this project as a tar.gz file

Requirements

  • jQuery
  • jQuery UI
  • Bootstrap (optional)

How to include

In your <head>-Tag:

<link href="css/bootstrap.min.css" rel="stylesheet"> <!-- OPTIONAL -->
<script src="js/jquery-1.9.1.min.js" type="text/javascript"></script>
<script src="js/jquery-ui-1.10.1.custom.min.js" type="text/javascript"></script>

<script src="js/jquery.etformvalidation.js" type="text/javascript"></script>

How to use

<script type="text/javascript">
    $(document).ready(function() {
        $("form").etformvalidation(/* options */);
    });
</script>

Options

  • bootstrap
    • true / false (default)
  • dateFormat
    • mm/dd/yy (default)
  • error
    • Please fill in the whole form.
