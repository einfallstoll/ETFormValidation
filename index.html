ETFormValidation

Easy to use jQuery plugin for validating forms.

View on GitHub Download as .zip Download as .tar.gz

What's ETFormValidation

ETFormValidation is a simple jQuery-Plugin which allows to validate forms as easy as possible!

What's required?

  • jQuery
  • jQuery UI (optional, required for date-validation)
  • Bootstrap (optional, required for Bootstrap-support)

Attributes (HTML5)

  • <input>, <select> & <textarea>
    • data-etfv="simple" field can't be empty
    • data-etfv="" Same as above
  • <input> with type="text"
    • data-etfv="date" checks for a valid date (requires jQuery UI!)
    • data-etfv="number" checks for a number and prevents other characters
    • data-etfv="5" adds a limitation to 5 characters (every number is possible)

Options (JS)

  • bootstrap: true | false enable Bootstrap
  • dateFormat: string jQuery UI Date Format
  • error: string displayed when the user didn't fill out all required fields

Getting Started

Either you download the required sources and link them in your head or you can just add this to your <head>-Tag: 

<!-- jQuery 1.9.1 -->
<script src="http://code.jquery.com/jquery-1.9.1.js"></script>

<!-- jQuery UI 1.10.2 -->
<script src="http://code.jquery.com/ui/1.10.2/jquery-ui.js"></script>

<!-- Bootstrap 2.3.1 -->
<link href="http://netdna.bootstrapcdn.com/twitter-bootstrap/2.3.1/css/bootstrap-combined.min.css" rel="stylesheet">
<script src="http://netdna.bootstrapcdn.com/twitter-bootstrap/2.3.1/js/bootstrap.min.js"></script>

<!-- ETFormValidation 0.1.0 -->
<script src="http://einfallstoll.github.com/ETFormValidation/js/jquery.etformvalidation.js"></script>

Now you need to create your <form>. You really need this "desgin": 

<form class="form-horizontal">
	<fieldset>
		<div class="control-group">
			<label class="control-label">Text</label>
			<div class="controls">
				<input type="text" />
			</div>
		</div>
	</fieldset>
</form>
This will create this <form>:

Now we want to say which fields are validated. <input>, <select> & <textarea> support a simple "required"-option with the attribute data-etfv="simple": 

<input type="text" data-etfv="simple" />
<input> with type="text" support some other validation-options:

  • data-etfv="date" checks for a valid date (requires jQuery UI!)
  • data-etfv="number" checks for a number and prevents other characters
  • data-etfv="5" adds a limitation to 5 characters (every number is possible)

Note: If you add the data-etfv-attribute the field is automatically marked as required! Maybe there will be another solution in the future.

Last, but not least, you have to add some JavaScript to activate the validation: 

<script type="text/javascript">
	$(document).ready(function() {
		$("form").etformvalidation();
	});
</script>
You may want add to add some options: 
<script type="text/javascript">
	$(document).ready(function() {
		$("form").etformvalidation({
			bootstrap: true,
			dateFormat:"mm.dd.yy",
			error:"You forgot to fill in the whole form."
		});
	});
</script>
